# MDX セキュリティルール

## 使用可能な Markdown 標準構文

| 構文 | 説明 | 例 |
|------|------|-----|
| 見出し | h1-h6 | `# 見出し1`, `## 見出し2` |
| 段落 | テキストブロック | 通常のテキスト |
| 強調 | 太字・斜体 | `**太字**`, `*斜体*` |
| リスト | 箇条書き・番号付き | `- 項目`, `1. 項目` |
| 引用 | ブロッククォート | `> 引用テキスト` |
| コード | インライン・ブロック | `` `code` ``, ` ```js ``` ` |
| リンク | ハイパーリンク | `[テキスト](URL)` |
| テーブル | 表組み | `| A | B |` |
| 水平線 | 区切り線 | `---` |
| 画像（Markdown） | 外部URL画像 | `![alt](URL)` |

## カスタムコンポーネント

使用可能なカスタムコンポーネントは MCP ツール `get_component_catalog` で取得すること。
サイト（ワークスペース）ごとに登録済みのコンポーネントが異なる。

## 禁止事項

以下の構文・パターンは**セキュリティ上の理由から使用禁止**です。

| 禁止項目 | 説明 |
|----------|------|
| `import` 文 | モジュールのインポートは不可 |
| `export` 文 | モジュールのエクスポートは不可 |
| JavaScript式 | `{() => {}}`, `{function}`, `{new}` 等 |
| `eval()` | 動的コード実行は不可 |
| `new Function()` | Function constructorは不可 |
| `<script>` タグ | スクリプトタグは不可 |
| イベントハンドラ | `onClick`, `onLoad` 等の `on*` 属性は不可 |
| `javascript:` URL | JavaScriptプロトコルは不可 |
| `data:` URL（src属性） | data URLはXSSベクターのため不可 |
| 未登録コンポーネント | `get_component_catalog` に存在しないコンポーネントは不可 |
